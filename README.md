# Classificação da Qualidade de Vinhos com Árvore de Decisão

Projeto desenvolvido utilizando Machine Learning para classificação da qualidade de vinhos com base em propriedades físico-químicas.

## Objetivo

Aplicar o algoritmo Árvore de Decisão para classificar vinhos em categorias de qualidade utilizando o dataset Red Wine Quality.

## Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Dataset

Red Wine Quality Dataset:
https://raw.githubusercontent.com/aniruddhachoudhury/Red-Wine-Quality/master/winequality-red.csv
